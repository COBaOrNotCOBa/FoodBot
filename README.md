# FoodBot

Телеграм-бот, который должен был рекомендовать блюда с помощью разных чат-ботов/моделей. В процессе разработки пробовал подключать несколько сторонних сервисов, но на момент работы с проектом мешали ограничения и необходимость дополнительной настройки VPN — поэтому проект остался экспериментальным.

## Что планировалось

- Рекомендации блюд по запросу пользователя
- Использование внешних чат-ботов / моделей для генерации советов
- Простой интерфейс через Telegram

## Что есть сейчас

- Базовая структура Telegram-бота
- Наброски логики рекомендаций
- Попытки интеграций с внешними сервисами

## Технологии

- Kotlin
- Telegram Bot API
- Интеграции с чат-ботами (конкретный набор зависит от текущего кода)

## Статус проекта

Эксперимент в области рекомендательных ботов. В текущем виде — черновик: может служить основой, если позже появится желание допилить интеграции и обойти ограничения по VPN.
